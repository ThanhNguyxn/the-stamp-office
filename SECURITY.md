# Security Policy

Even though this project is currently documentation-first, we take security seriously.

## Reporting a Vulnerability
If you discover a security vulnerability, please report it by email:
**security@project.invalid**

Please include:
- A clear description of the issue
- Steps to reproduce (if applicable)
- Impact assessment (what could happen)
- Any relevant logs/screenshots (redact sensitive info)

## Response
We will acknowledge receipt of your report and investigate as soon as we can.  
We do not provide guaranteed timelines, but we aim to respond reasonably and transparently.
